Dev Utilities

A collection of free online tools for developers. Everything runs in your browser.

CSS Compressor

Minify and compress CSS code into multi-line or single-line format. Remove whitespace and comments.
SQL Formatter

Format and beautify SQL queries with proper indentation. Supports MySQL and T-SQL dialects.
Markdown to HTML

Convert Markdown text to clean, formatted HTML with proper indentation and one-click copy.
HTML Entity Encoder

Encode special characters like < and > into their HTML entity equivalents for safe embedding.
Random String Generator

Generate random strings with configurable length and quantity. Choose letters, numbers, or special characters.
Image Splitter

Split an image into equal horizontal slices. Drag-and-drop upload, download all slices as ZIP.
FFMpeg

Generate FFmpeg trim and crop commands for Windows. Set video and audio bitrates, define start/end times per clip, and copy ready-to-run commands directly to your terminal
All tools run entirely in your browser. No data is sent to any server.
